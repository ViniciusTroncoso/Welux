# Spec: Copy QI 200 — Welux Landing Page

**Status:** Aprovada para implementação  
**Data:** 2026-06-03  
**Objetivo:** Elevar o posicionamento de "agência de IA" para "Software House / Consultoria High-Ticket de Engenharia de IA" — copy orientada a CEO/Diretor, vocabulário de board executivo, lógica de venda negativa (takeaway selling).

---

## Princípios da copy QI 200

1. **Venda Negativa (Takeaway):** Vetamos o projeto se o ROI não fechar. Avisamos se o cliente não tiver maturidade. Isso destroi resistência e gera confiança cega.
2. **Vocabulário de board executivo:** Maturidade de dados, Infraestrutura, Risco Mitigado, Raio-X Operacional, Lucro Líquido, Viabilidade Técnica.
3. **Contraste brutal:** Separa da "agência que adicionou IA no nome" — posiciona como força-tarefa de engenharia.
4. **Zero jargão de marketing:** Sem "soluções inovadoras", "ecossistema inteligente", "transformação digital".

## Critérios de aceite

- [ ] Zero palavras: "soluções", "inovador", "transformação digital", "ecossistema"
- [ ] Todo CTA usa "Viabilidade" ou "Análise" — nunca "grátis" ou "gratuita" (posicionamento high-ticket)
- [ ] Qualificador de faturamento presente abaixo do CTA principal da Hero
- [ ] Venda negativa explícita em pelo menos 2 pontos (Method passo 2 + FinalCta subtítulo)
- [ ] "Software House" aparece pelo menos 3x (Hero sub, Services título, About parágrafo 2)
